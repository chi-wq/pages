# pages

本仓库是托管在 GitHub Pages 上的 **Jekyll 博客/笔记站点**的源代码。

- 站点地址: https://chi-wq.github.io/pages/

## 目录结构

- `_posts/`：博客文章（按日期排序）
- `_entries/`：笔记/条目（collection），可按需整理为 wiki 风格
- `assets/`：CSS / JS / 图片等静态文件

部分页面使用 front matter 中的
`wiki_key` / `wiki_order` / `wiki_root` 来实现左侧边栏的树形导航（仅在需要的页面中设置）。
